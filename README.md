# Simple Calculator

## Description
A simple command-line calculator developed using Python. This application allows users to perform basic arithmetic operations by entering two numbers and selecting the desired operation from a menu.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- User-friendly command-line interface

## Technologies Used
- Python 3

## How to Run
```text
python calculator.py
```
